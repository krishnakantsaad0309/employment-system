# Employment System - Job Portal

A complete job portal with Admin, Employer, and Job Seeker roles.

## Requirements

- **Node.js**: v14 or higher
- **MongoDB**: v4.4 or higher
- **npm**: v6 or higher

## Quick Setup (From ZIP File)

### Step 1: Extract ZIP File
Extract the project ZIP file to your desired location.

### Step 2: Install MongoDB
Make sure MongoDB is installed and running on your system.

**Windows:**
```bash
net start MongoDB
```

### Step 3: Setup Backend

Open terminal/command prompt in project folder:

```bash
cd backend
npm install
npm run seed
npm run dev
```

Backend will run on: `http://localhost:5000`

### Step 4: Setup Frontend

Open **NEW** terminal/command prompt in project folder:

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on: `http://localhost:5173`

### Step 5: Open Browser

Go to: `http://localhost:5173`
